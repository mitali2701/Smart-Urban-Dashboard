# 🚦 Smart Urban Dashboard

A **Smart Urban Dashboard** to monitor and predict key urban metrics:
- 🚦 Traffic Prediction  
- 🌫️ Pollution Forecast  
- 🛣️ Speed Limit Check  
- 🚧 Accident Hotspots  
- 🌐 Live Weather & AQI Updates  
- 🔔 Auto-Refresh & Notifications for Alerts!
